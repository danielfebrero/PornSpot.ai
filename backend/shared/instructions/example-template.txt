Instructions for using instruction templates:

1. **File naming**: Use descriptive names ending with .txt
2. **Variable placeholders**: Use {{variableName}} format for dynamic content

   - Include examples when helpful
   - Define the expected output format
   - Consider the model's strengths and limitations
